# Hand2Hand 🛍️♻️

> A sustainable marketplace for renting and selling second-hand goods.

Hand2Hand is a cross-platform mobile application built with **Flutter** and **Supabase** that enables users to rent, buy, and sell second-hand products in a secure and user-friendly environment. The platform promotes sustainable consumption by extending the lifecycle of everyday items and reducing unnecessary waste.

---

## 📖 Overview

Every year, countless usable products such as furniture, electronics, books, and household items remain unused or are discarded. Hand2Hand provides a trusted platform where users can list these items for rent or sale, helping others save money while encouraging a circular economy.

---

## ✨ Features

### 🛒 Marketplace

* Buy and sell second-hand products

### 👤 User Management

* Secure authentication
* User profile management
* NID-based identity verification

* Live notifications

### 📍 Location Services

* Location-based listings

### 🛡️ Admin Panel

* User verification
* Product moderation

* Platform administration

## 🚀 Getting Started

### Prerequisites

* Flutter SDK
* Dart SDK
* Android Studio / VS Code
* Supabase Project

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/hand2hand.git

# Navigate to project
cd hand2hand

# Install dependencies
flutter pub get

# Run the application
flutter run
```
